🧪 Formulario Genérico (Prueba de Concepto)

Formulario de prueba para la funcionalidad "Formulario Genérico". Usá la extensión → pestaña "Formulario Genérico" → cargá Excel → mapeá → ejecutá.

¿Cómo probar? Abrí el popup → pestaña "Formulario Genérico" → cargá un Excel con columnas NOMBRE, CEDULA, CORREO, TELEFONO, CARGO → "Detectar Campos" → mapeá → "Ejecutar".

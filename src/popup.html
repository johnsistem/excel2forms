📖 Quick Start Guide

How it works:

  1. Load Excel - Drag your .xlsx or .csv file into the upload zone
  2. Detect Fields - Go to the form page and click "Detect Fields"
  3. Map Columns - Assign each Excel column to a form field
  4. Set Save Button - Select the submit button (usually detected automatically)
  5. Execute - Choose Manual or Auto mode and run

⚠️ Recommended: Use Manual mode the first time to see how the extension fills the form. Once confirmed, switch to Auto for bulk execution.

💡 Errors are logged per row. A summary with download report appears at the end.

📋 Excel Format

Plain format (one row = one record):

FULL NAME EMAIL DATE OF BIRTH COUNTRY INTERESTS
James Wilson james@email.com 1991-04-12 United States Sports, Technology

The first row must contain the column names. Each column is assigned to a form field.
